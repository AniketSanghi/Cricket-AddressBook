# Cricket_AddressBook
Can Edit, Delete, Insert, Scrap data in the Cricket Address Book

Bash Script (Demo 1)

CS252 - Computer Laboratory II

Prof. Nisheeth Shrivastav

## How to Run

- Change Permissions

        chmod 700 ./scrapData.sh
        chmod 700 ./run.sh
        
- Scrap data from CricInfo

        ./scrapData.sh
        
- Run the interactive shell

        ./run.sh
